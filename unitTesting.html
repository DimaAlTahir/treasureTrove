Unit Testing(white box testing)

I used Unit Testing to verify if my internal coding and infrastructure meets the specifications. I accomplished it by testing the flow of inputs and output of 3 components of the Application

For the leaderboard i used various inputs like negative numbers, positive numbers, characters, symbols, and numbers with fractional parts with the options completed or not

For the score i also used various inputs like negative numbers, positive numbers, characters, symbols, and numbers with fractional parts with the options Has prize and Sorted

For the questions i was testing each type of Questions with the options Geolocation and skip. Lastly i tested whether or not the option completed would indeed work on all of the types and redirect the user to the leaderboard

Examples of these tests that represent each type of input were recoded on tables. Based on my results i would say that my application meets the requirements even though the server was sending unexpected information
